  • 接口修饰符

Java接口中的方法并非没有修饰符，而是省略了。在Java的接口中，只有常量和抽象方法。所有的数据域均用 public static final 修饰，所有的方法都用 public abstract 修饰，所以修饰符是可以省略的。

Java 8 中，接口中可以定义默认方法和静态方法了。
